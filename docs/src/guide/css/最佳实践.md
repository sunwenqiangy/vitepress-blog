# 最佳实践

## 注释的写法

  ```html
  /* Footer */
  内容区
  /* End Footer */
  ```





##  CSS书写

- 布局定位属性
  - display

  - position

  - float

  - clear

  - visibility

  - overflow

- 盒子模型+背景
  - width
  - height
  - margin
  - padding
  - border
  - background
- 文本内容属性
  - color
  - font
  - text-decoration
  - text-align
  - vertical-align
  - line-height
- 点缀属性
  - content
  - cursor
  - border-radius
  - box-shadow
  - text-shadow
  - background:linear-gradient

注意：
	开发中推荐**多用类+后代**，但不是层级越多越好，一个选择器中的类选择器的个数推荐**不要超过3个**。
